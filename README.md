# AAPT2ForNet

A .NET library to dump APK with AAPT2

## Support Platform
- .NET Framework 4.5+
- .NET Standard 2.0+
- .NET Core 5.0 (UAP 10.0.15138.0+)
- .NET Core App 2.0+
